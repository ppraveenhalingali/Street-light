# Smart Street Light Control System using Raspberry Pi 4

## 📌 Overview
The Smart Street Light Control System is an IoT-based embedded project developed using Raspberry Pi 4. The system automatically controls street lights based on environmental light conditions and allows real-time monitoring and control through a web application.

An LDR sensor detects ambient light intensity, and the Raspberry Pi processes the sensor data to automatically turn street lights ON during nighttime and OFF during daytime. Users can also monitor and control the street lights remotely using a web dashboard.

This project demonstrates the use of embedded systems, IoT, and web technologies for smart city infrastructure and energy-efficient lighting.

## 🧠 Working Principle
1. The LDR sensor detects ambient light intensity.
2. Raspberry Pi reads the sensor data through GPIO.
3. When the environment becomes dark, the system turns the street light ON.
4. During daytime, the street light automatically turns OFF.
5. The system sends data to the web application for real-time monitoring.
6. Users can also manually control the lights through the web dashboard.
